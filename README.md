


# ArunBus Website

## Description

The ArunBus Website is a web application designed to provide users with an easy and efficient way to access bus schedules, routes, and other relevant information related to ArunBus services. Built using modern web technologies, the platform combines a user-friendly interface with robust backend functionality to enhance the travel experience for passengers.

### Technologies Used

- **HTML**: Structuring the web pages and content.
- **CSS**: Styling the website to ensure a responsive and visually appealing layout.
- **Python**: Backend logic and server-side operations.
- **Django**: Web framework used to develop the backend, manage data, and handle user interactions.

### Features

- View bus schedules and routes.
- Real-time updates on bus statuses.
- Administrative panel for managing schedules and routes.
- Responsive design for accessibility on various devices.

### Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/arunbus-Website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd arunbus-Website
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run migrations:
   ```bash
   python manage.py migrate
   ```
5. Start the development server:
   ```bash
   python manage.py runserver
   ```
6. Open your browser and go to `http://127.0.0.1:8000/` to see the website in action.
